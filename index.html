¿Quienes somos?

Somos un grupo de programadores basados en Malaga, con una pasion para crear aplicaciones web funcionales y de calidad

Las tecnologias que mas utilizamos en nuestro desarrollo son las siguientes

  • React para el frontend
  • PHP solo, o con un framework como Laravel para el Backend
  • MySQL o PostgreSQL para Bases de Datos

Nuestro programador principal es Charles David Crane, aqui puedes acceder a su LinkedIn para ver la experiencia y formacion que va a traer a tus proyectos

Todos nosotros contamos con mucha experiencia programando con varias tecnologias, realizando mantenimientos o montando aplicaciones web desde 0

A parte de todo lo dicho hasta ahora, tenemos formacion de ciberseguridad en entornos web, entonces puedes asegurarte de que tu aplicacion estará seguro
